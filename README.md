# voice-controlled-virtual-assistant
Features:
1.Speech Recognition (voice input)
2.AI Responses using OpenAI API
3.Text-to-Speech output
4.Open websites (YouTube, Google, etc.)
5.Tell current time Continuous conversation (chat memory)
6.Save AI responses to files

Tech Stack:
Python
OpenAI API
SpeechRecognition
Pyttsx3
NumPy

Description:
This project is a voice-controlled AI assistant built using Python that can interact with users through natural speech. It uses speech recognition to understand voice commands and responds using text-to-speech, creating a hands-free conversational experience. The assistant can perform tasks such as opening websites, telling the current time, and answering questions by integrating with the OpenAI API for intelligent, human-like responses.

Designed as a modular and beginner-friendly project, it demonstrates how to combine multiple technologies like speech processing, automation, and artificial intelligence into a single application. The system also maintains a simple conversational memory and can save AI-generated outputs for later use. This project serves as a strong foundation for building more advanced virtual assistants with features like wake-word detection, GUI interfaces, and personalized interactions.
This is inspired by Jarvis Virtual assistant
